Dans le répertoire de l'éxécutable, lancez les commandes suivante
1.	'make' 
2.	`./automaton nom_du_fichier.aut` 
3. 	entrer un mot du langage 

--> les noms des fichiers des automates présent dans le répertoires sont :
	- word.aut
	- word_bis.aut
	- dyck.aut 
	- arith.aut

** Pour input un nouveau mot il faut recommencer à l'étape 2 
(voir le 3ème point de Limites du Programme dans le rapport pour savoir 
pourquoi cela propose pas automatiquement d'entrer un nouveau mot) 
**

** Le Dossier Annexe contient quelque fichier comme mistakes.c qui énumére les bouts de 
code qui s'avérait faux
